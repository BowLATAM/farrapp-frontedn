## Get Started

To get started consider:

### Requirements

You will need to install:

#### NodeJs

- [NodeJs](https://nodejs.org/es/download/)

#### Yarn

- [Yarn](https://yarnpkg.com/en/docs/install)

### Set up project

- install dependencies:

  ```bash
  yarn install
  ```

- run application locally:

  ```bash
  yarn start
  ```

- run tests:

  ```bash
  yarn test
  ```

- create a build:
  ```bash
  yarn build
  ```
